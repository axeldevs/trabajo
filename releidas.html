Generación de rutas para Releidas

SICOM NUEVO

  1. Ingresamos con claves de acceso RELEI:
    • LOGIN: RELEI
    • YES
    • PASS: CFEDIS-Db000 | Db000-CFEDIS
  2. Ingresamos los datos de cada ruta. Ejemplo:
    • agencia: [DB10J] (ddzza)
    • Ciclo: [26]
    • Pob: [B3]
    • Ruta: [261]

NIVEL 0 (CERO)

  1. opción 5) Reportes
  2. Opcion 2) Usuarios sin toma de lecturas por Tarea
  3. Se escribe cada tarea, esto se manda a SCRICFE. Se junta toda la informacion en un WORD y se imprime
  4. Se procede a generar la tarea de releidas como una generacion cualquiera, solo que se esta vez se seleccionan la CVE R (releidas)
  5. Nota * Para integrar las releidas primero hay que cambiar el estado de las rutas con la CVE 150 a 00 y repetir el proceso de union de tareas
