Generación de tareas para toma de lectura

1. Tener las cifras control del ciclo a generar

En nivel 1 (UNO)

  1. Opción 03 - Cambios Directos ( E-19 )
  2. Como parametro Ponemos el número de cuenta hasta la ruta y terminar con ceros
    Ejemplo: 32DB10A013210000
  3. Dar F1 para saber la CLAVE que pondremos para toma de lectura dependiendo el MES A FACTURAR
  4. Al poner la clave correspondiente pedirá el AÑO A FACTURAR: AAAA
  5. Te preguntará si deseas calendario de fechas reales. Elegimos que no (N)
  6. Ponemos la fecha de toma de lectura: AAAAMMDD
  7. Pregunta si es correcta la modificación. Ponemos que si (S)
  8. Volvemos a poner el año a facturar: AAAA
  9. De nuevo preguntará si deseas calendario de fechas reales. Ahora elegimos que si (S)
  10. Volvemos a poner la fecha de toma de lectura: AAAAMMDD
  11. Pregunta si es correcta la modificación. Ponemos que Si (S)

En nivel 0 (CERO)

  1. Opcion 01 - Selección Rutas Toma de Lect.
  2. Ponemos la fecha en que se tomará el ciclo a generar: AAAAMMDD
  3. Seleccionamos cada una de las rutas del ciclo a generar y damos ENTER
  4. Al seleccionar todas las rutas presionamos F8. Preguntará si procede a la generación de archivos para lectura. Ponemos Si (S)
  5. Esperamos...
  6. Pondrá la leyenda de proceso terminado correctamente. Damos ENTER
  7. Regresamos a menu principal Nivel 0 (CERO)
  8. Opcion 02 - Explotación Tareas LIRE
  9. Opcion 01 - Generación Tareas/Reporte de cortes/Lecturas/Ciclo limpio
  10. Esperamos...
  11. Si sale la leyenda de Archivo de lecturas sin movimientos damos ENTER y seguimos esperando.
  12. Seleccionamos las rutas para toma de lectura una por una. Las rutas para Lecturas(Tienen CVE L). Las rutas para Releidas(Tienen CVE R)
  13. Damos ENTER hasta tarea y escribimos la tarea en la que se leerá cada ruta. En sello y F-PROP damos ENTER
  14. Bajamos con la flecha (↓) y repetimos el proceso hasta hacerlo con todas las rutas
  15. Al tenerminar presionar F8
  16. Luego F1 para seleccionar enviar a KAVI
  17. Escribimos el RPE del Auxiliar que tomará cada ruta. Sacar CAPTURA DE PANTALLA y esperamos...
  18. Una vez que termine de procesar el sistema presionamos ENTER y podremos cargar las tareas en las TPs
